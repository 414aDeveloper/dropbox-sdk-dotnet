Dropbox.Api.Babel Namespace

[This is preliminary documentation and is subject to change.]

Contains the definitions for the interfaces used for encoding and transporting babel api objects. This also includes concrete implementations of those interfaces.
Classes
  ClassDescription
Empty
An empty object used when a route doesn't have one or more of the request, response, or error types specified.
JsonDecoder
An IDecoder implementation that decodes IEncodableT objects from a JSON string.
JsonEncoder
An IEncoder implementation that encodes IEncodableT objects to a JSON string.
Util
Contains utility extension methods.
Interfaces
  InterfaceDescription
IDecoder
Interface that is used to decode objects that implement the IEncodableT interface.
IDecoderContext
A context used to get decoded field values from an encoded object.
IDownloadResponseTResponse
Used to encapsulate both the response object and the response body from a download operation.
IEncodableT
Interface implemented by all objects generated by the babel API generator. This is used to support transport serialization of objects.
IEncoder
Interface that is used to encode objects that implement the IEncodableT interface.
IEncoderContext
A context used to set field values for an encoded object.
