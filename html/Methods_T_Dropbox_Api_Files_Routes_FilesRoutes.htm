CopyAsync(RelocationArg)

Copy a file or folder to a different destination in the user's Dropbox.

If the source path is a folder all its contents will be copied. The destination path must not yet exist.

CopyAsync(String, String)

Copy a file or folder to a different destination in the user's Dropbox.

CreateFolderAsync(String)

Create a folder at a given path.

No file or folder may exist at the path. The parent folder will be created if it does not already exist (and so on). If the parent exists it must be a folder (and the same for any ancestor). If an ancestor is a shared folder it must have write access.

ListFolderGetLatestCursorAsync(ListFolderArg)

A way to quickly get a cursor for the folder's state. Unlike ListFolderAsync(ListFolderArg), ListFolderGetLatestCursorAsync(ListFolderArg) doesn't return any entries. This endpoint is for app which only needs to know about new files and modifications and doesn't need to know about files that already exist in Dropbox.
