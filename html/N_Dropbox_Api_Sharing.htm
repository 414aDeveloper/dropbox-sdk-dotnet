Dropbox.Api.Sharing Namespace

[This is preliminary documentation and is subject to change.]

This namespace contains endpoints and data types for creating and managing

shared links.

Classes
  ClassDescription
AccessType

The access type fo this shared folder. Each type is associated with specific access permission.

AccessTypeEditor

The member can both view and edit the shared folder.

AccessTypeOther

An unknown access type.

AccessTypeOwner

The member is the owner of the shared folder. The user can both view and edit the shared folder.

AccessTypeViewer

The member can only view the shared folder.

BasicSharedFolderMetadata

The metadata which includes basic information about the shared folder.

CollectionLinkMetadata

Metadata for a collection-based shared link.

CreateSharedLinkArg

Arguments for CreateSharedLinkAsync(CreateSharedLinkArg).

FullSharedFolderMetadata

The full metadata for the shared folder which includes user and group membership.

GetSharedFolderArgs

Arguments for GetSharedFolderAsync(GetSharedFolderArgs).

GetSharedLinksArg

Arguments for GetSharedLinksAsync(GetSharedLinksArg).

GetSharedLinksResult

Result for GetSharedLinksAsync(GetSharedLinksArg).

GroupInfo

The information about a group. Group is a way to manage a list of users who need same access permission to the shared folder.

GroupMembershipInfo

The information about a group member of the shared folder.

LinkMetadata

Metadata for a shared link. This can be either a PathLinkMetadata or CollectionLinkMetadata.

ListSharedFoldersArgs

Arguments for ListSharedFoldersAsync(ListSharedFoldersArgs).

ListSharedFoldersResult

Result for ListSharedFoldersAsync(ListSharedFoldersArgs).

MembershipInfo

The information about a member of the shared folder.

PathLinkMetadata

Metadata for a path-based shared link.

PendingUploadMode

Flag to indicate pending upload default (for linking to not-yet-existing paths).

PendingUploadModeFile

Assume pending uploads are files.

PendingUploadModeFolder

Assume pending uploads are folders.

SharedFolderAccessError

There is an error accessing the shared folder.

SharedFolderAccessErrorHasLeft

The user has left the shared folder already thus is no longer able to access it.

SharedFolderAccessErrorInvalidId

This shared folder ID is invalid.

SharedFolderAccessErrorIsAppFolder

The folder is an app folder and cannot be shared.

SharedFolderAccessErrorIsRoot

The folder is a root folder and cannot be shared.

SharedFolderAccessErrorIsTeamSharedFolder

The folder is a team shared folder and the user cannot access it.

SharedFolderAccessErrorNotMember

The user is not a member of the shared folder thus cannot access it.

SharedFolderAccessErrorRequireOwner

The user needs to be the owner to access the shared folder.

SharedFolderMetadata

The base type for shared folder metadata.

SharedLinkPolicy

Policy governing who can links be shared with.

SharedLinkPolicyAll

Links can be shared with anyone.

SharedLinkPolicyMembersOnly

Links can only be shared among members of the shared folder.

SharedLinkPolicyOther

An unknown shared link policy.

UserInfo

The information about a user.

UserMembershipInfo

The information about a user member of the shared folder.

Visibility

Who can access a shared link. The most open visibility is VisibilityPublic. The default depends on many aspects, such as team and user preferences and shared folder settings.

VisibilityOther

An unknown restriction is in place.

VisibilityPassword

A link-specific password is required to access the link. Login is not required.

VisibilityPublic

Anyone who has received the link can access it. No login required.

VisibilitySharedFolderOnly

Only members of the shared folder containing the linked file can access the link. Login is required.

VisibilityTeamAndPassword

Only members of the same DfB (Dropbox for Business) team who have the link-specific password can access the link.

VisibilityTeamOnly

Only members of the same DfB (Dropbox for Business) team can access the link. Login is required.
